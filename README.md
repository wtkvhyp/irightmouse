## 它到底能做什么

### 右键新建文件

这是用户用得最多的功能。TXT、Word、Excel、PPT、Markdown、Pages、Keynote、PSD、Ai……十几种格式都可以直接右键新建。还支持自定义模板，比如你有一个每次都要用到的合同模板，直接放进去，以后右键一点就生成了，再也不用复制粘贴旧文件再改名。

### 剪切、粘贴文件

macOS 的一个历史遗留问题就是没有文件剪切功能。超级右键把这个补上了，还贴心地支持两种模式：剪切时隐藏选中的文件（像 Windows 那样变灰）或者不隐藏——看你习惯。

### 发送 / 复制文件到指定位置

不用打开访达一层层找目标文件夹，直接右键 → 移动到 / 复制到，在弹出的列表里点目标位置，搞定。如果常用的文件夹比较固定，设置好以后效率真的高很多。

### 常用目录快捷直达

跟上面类似，把你高频访问的文件夹加进去，在任何路径下右键都能直接跳过去。做设计的、写代码的、处理文档的，基本都会有几个天天要打开的目录，这个功能对这类用户来说是救星。

### 自定义文件夹图标

这个算是彩蛋功能。用默认的蓝色文件夹找起来确实费劲，换上颜色不同的图标，视觉上一下就清晰了。Better365 还提供了大量图标资源可以下载。

### 开发者快捷入口

在文件夹上右键，可以直接用终端、iTerm2、VSCode、Sublime Text、PyCharm、Typora 等几十种工具打开当前路径或文件。支持的编辑器和 IDE 列表相当长，基本覆盖了主流开发工具。

### 工具箱

彻底删除（一步跳过废纸篓）、拷贝路径、隐藏/显示文件、隔空投送、查看文件 MD5/SHA 信息、图片转为 WebP/ICNS/PNG/JPG……这些零散但实用的小功能都集中在工具箱里，用到的时候不用单独找工具，右键就有。
